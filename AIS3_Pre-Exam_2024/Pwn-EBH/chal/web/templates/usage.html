EBH Control Panel

使用說明

這題不需要去分析 Web 的部分。Web 的部分簡單來講就是讓你上傳一個 binary，然後把這個 binary 在 QEMU 裡面跑的結果回傳給你。

使用範例
  1. 寫好一個檔案叫做 example.c <- 這是鏈結
  2. 在 x86 架構的電腦編譯 gcc -static example.c -o example
  3. 進入 EBH Control Panel 後按 Calculate 計算 PoW
  4. 等待計算完成後網頁會自動把 PoW 的答案填入 Answer 欄，按下 Submit
  5. 接著上傳 example 並 Submit
  6. 等待 Challenge 跑完後網頁會顯示結果，正常情況下結果應該會顯示 Hello EBH !
  7. 如果想要重新上傳一個 binary 的話按下 Try Again 就可以重新來一遍了
  8. 如果 30 分鐘都沒有重新整理或 Try Again 的話 uid 就會過期，所以如果覺得怪怪的就重新整理看看
